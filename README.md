# デジタルサイネージ モックアップ

クライアント向け 画面デザイン案（静的モック）

## 📂 構成

```
digital-signage/
├── index.html                    # モック一覧ページ
├── css/
│   └── common.css               # 共通スタイル
└── screens/
    ├── vertical/                # 縦置きモニター用
    │   ├── 01_行動予定表.html
    │   └── 02_施工サイクル表.html
    ├── horizontal/              # 横置きモニター用
    │   ├── 01_協力業者一覧.html
    │   └── 02_月間行事予定表.html
    └── poster-patterns/         # ポスター表示パターン
        ├── pattern_A_フルスクリーン切替.html
        └── pattern_B_分割表示.html
```

## 🖥️ 画面一覧

### 縦置き（1080×1920）
| 画面名 | 説明 |
|--------|------|
| 行動予定表 | 氏名・行先・使用車両・帰社予定時刻を一覧表示 |
| 施工サイクル表 | 作業内容とステータスを表示 |

### 横置き（1920×1080）
| 画面名 | 説明 |
|--------|------|
| 協力業者作業一覧 | 当日入場の協力業者をカード形式で表示 |
| 月間行事予定表 | 日付・曜日・予定をカレンダー形式で表示 |

### ポスター表示パターン
| パターン | 説明 |
|----------|------|
| A: フルスクリーン切替 | 表とポスターを交互に切り替え（推奨） |
| B: 分割表示 | 画面を上下分割して同時表示 |

## 🚀 確認方法

1. `index.html` をブラウザで開く
2. 各画面リンクをクリックして確認

- [ ] ポスター表示方式（A or B）
